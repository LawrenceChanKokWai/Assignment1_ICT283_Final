\doxysubsection{Detailed Description}
A class representing a map consisting array of Vectors of sensor data. 

This class provides functionality to manage sensor data organized by month and year. It supports operations such as insertion of sensor data, computation of mean and standard deviation, and display of aggregated data for specific months or years.
