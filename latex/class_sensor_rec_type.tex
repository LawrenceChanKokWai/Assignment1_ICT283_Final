\doxysubsection{Detailed Description}
Represents a sensor record type with date, time, wind speed, temperature, solar radiation components. 

The \mbox{\hyperlink{class_sensor_rec_type}{Sensor\+Rec\+Type}} class encapsulates information about a specific sensor record, including date, time, wind speed, solar radiation, and temperature. It provides methods to retrieve and set these components.
